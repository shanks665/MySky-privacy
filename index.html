1. はじめに

このプライバシーポリシーは、MySky.（以下「当社」または「当アプリ」といいます）が、ユーザーの個人情報をどのように収集、使用、保護、共有するかについて説明するものです。当アプリをご利用いただくことにより、ユーザーは本プライバシーポリシーに記載された情報の収集と利用に同意したことになります。

当社は、ユーザーのプライバシーを尊重し、適用される全てのデータ保護および個人情報保護に関する法律および規制を遵守することを約束します。これには、個人情報保護法、GDPR（EU一般データ保護規則）、CCPA（カリフォルニア州消費者プライバシー法）などが含まれます。

2. 収集する情報

2.1 ユーザーが提供する情報

当アプリでは、以下のようなユーザーから直接提供される情報を収集する場合があります：

  • アカウント情報：ユーザー名、メールアドレス、パスワード
  • プロフィール情報：氏名、プロフィール写真、生年月日、性別、居住地
  • ユーザーコンテンツ：テキスト投稿、コメント、写真、動画など、アプリ内で作成、共有する内容
  • 通信内容：他のユーザーとのメッセージやコミュニケーション
  • イベント情報：作成または参加するイベントに関する情報

2.2 自動的に収集される情報

当アプリを利用する際、以下の情報が自動的に収集される場合があります：

  • デバイス情報：デバイスの種類、オペレーティングシステム、デバイス識別子
  • ログ情報：IPアドレス、ブラウザタイプ、アクセス日時、閲覧したページ
  • 位置情報：GPSデータ、Wi-Fiアクセスポイント情報（位置情報の許可がある場合のみ）
  • 使用状況データ：アプリの機能やサービスの利用方法、ユーザーインタラクション
  • クッキーとトラッキング技術：セッションデータ、利用統計、分析情報

2.3 第三者から取得する情報

以下のような第三者から情報を取得する場合があります：

  • ソーシャルメディア：ソーシャルメディアアカウントを使ってログインすると、それらのプラットフォームから一部のプロフィール情報を取得します
  • 分析パートナー：サービス利用に関するデータをトラッキングするパートナーからの情報
  • 広告パートナー：キャンペーンの効果測定のための情報

3. 情報の利用方法

収集した情報は以下の目的で利用します：

3.1 サービス提供のため

  • アカウント作成と維持
  • ユーザー認証と本人確認
  • カスタマーサポートの提供
  • ユーザー間のコミュニケーション機能の実現
  • イベント作成・参加機能の実現

3.2 サービス改善のため

  • ユーザー体験の向上とパーソナライズ
  • アプリの機能とパフォーマンスの分析
  • バグの特定と修正
  • 新機能の開発と評価

3.3 コミュニケーションのため

  • サービスに関する通知の送信
  • アップデートや新機能のお知らせ
  • カスタマーサポートへの対応
  • 利用規約やポリシーの変更通知

3.4 セキュリティと法的要件のため

  • アプリとユーザーの安全保護
  • 不正行為の検出と防止
  • 紛争解決と権利行使
  • 法的義務の遵守

4. 情報の共有

当社は、以下の場合にユーザー情報を第三者と共有することがあります：

4.1 ユーザーの同意がある場合

ユーザーの明示的な同意がある場合、指定された第三者と情報を共有することがあります。

4.2 サービス提供者

当社に代わってサービスを提供する第三者と情報を共有する場合があります：

  • クラウドホスティングプロバイダー（例：AWS、Google Cloud）
  • データ分析サービス（例：Google Analytics、Firebase）
  • カスタマーサポートツール
  • 決済処理サービス

これらのサービス提供者は、当社の指示に従って情報を処理し、本プライバシーポリシーに準拠する義務があります。

4.3 法的要件

以下の場合、ユーザー情報を開示することがあります：

  • 法律、規制、法的手続き、政府の要請に従う必要がある場合
  • 詐欺、セキュリティ問題、技術的問題を検出・防止・対処する必要がある場合
  • 当社、ユーザー、第三者の権利、財産、安全を保護するために必要な場合
  • 企業買収、合併、資産売却などの取引に関連して必要な場合

4.4 匿名化・集計情報

当社は、個人を特定できない形に匿名化または集計した情報を、マーケティング、広告、調査目的で第三者と共有することがあります。

5. データセキュリティ

当社は、収集した個人情報を保護するために、業界標準の技術的・組織的セキュリティ対策を実施しています：

  • データの暗号化：転送中および保存時のデータを暗号化
  • アクセス制限：必要最小限の従業員のみがユーザーデータにアクセス可能
  • 定期的なセキュリティ評価：脆弱性スキャンやセキュリティ監査の実施
  • インシデント対応計画：データ侵害が発生した場合の対応手順の整備

しかし、インターネット上またはデータストレージシステムでの100％の安全性は保証できません。そのため、当社は最高レベルのセキュリティを維持するよう努めますが、絶対的な安全性を保証するものではありません。

6. ユーザーの権利

ユーザーには、個人情報に関して以下の権利があります：

  • アクセス権：当社が保持するあなたの個人情報にアクセスする権利
  • 訂正権：不正確または不完全な個人情報の訂正を要求する権利
  • 削除権（忘れられる権利）：特定の条件下であなたの個人情報の削除を要求する権利
  • 処理制限権：特定の状況下で個人情報の処理を制限する権利
  • データポータビリティの権利：あなたのデータを構造化された形式で受け取り、別のサービスに転送する権利
  • 異議申立権：特定の処理に対して異議を唱える権利
  • 同意撤回権：同意に基づく処理に対して、いつでも同意を撤回する権利

これらの権利を行使するには、下記の「お問い合わせ」セクションに記載された連絡先までご連絡ください。当社は、適用法に従い、合理的な期間内にご要求に対応いたします。

7. 子どものプライバシー

当アプリは、13歳未満の子どもを対象としていません。当社は、13歳未満の子どもから意図的に個人情報を収集することはありません。13歳未満の子どもが個人情報を提供したことが判明した場合、当社はできるだけ速やかにその情報を削除するための措置を講じます。13歳未満のお子様の情報が当社に提供されたと思われる場合は、下記の連絡先までご連絡ください。

8. 国際的なユーザー

当アプリは日本で運営されていますが、世界中のユーザーにサービスを提供しています。当社のサーバーは日本および他の国に所在している場合があり、あなたの情報はこれらの国で処理・保存される可能性があります。これらの国のデータ保護法は、あなたの居住国のものとは異なる場合があります。

当アプリを利用することにより、ユーザーはこの国際的なデータ転送に同意したものとみなされます。当社は、どの国で処理されるかにかかわらず、本プライバシーポリシーに従ってユーザーの情報を保護するために適切な安全対策を実施します。

9. ポリシーの変更

当社は、必要に応じて本プライバシーポリシーを更新することがあります。最新のポリシーは常にこのページで公開され、重要な変更がある場合は、アプリ内通知または登録されたメールアドレスへの連絡などの方法でユーザーに通知します。

ポリシーの変更後も引き続きアプリを利用することで、ユーザーは更新されたポリシーに同意したものとみなされます。定期的に本ページを確認して、最新のプライバシープラクティスについて情報を得ることをお勧めします。
